1.	В новый проект копируем файлы package.json, gulpfile.js и папку #src.
2.	Для быстрой установки всех плагинов в новый проект в терминале набираем npm i
3.	Установить converter командой - npm install webp-converter@2.2.3 --save-dev
4.	Шрифты копировать в папку fonts в #src в формате ttf
5.	WEBP-CSS выдает ошибку если в названии файла картинки есть пробелы и/или кириллица

	В любой непонятной ситуации иди смотреть видос о настройке Gulp
